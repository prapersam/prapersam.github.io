Servidor Web en Windows con IIS

Introducción al Servidor Web IIS

1. Configuración servidor + Instalación Servidor IIS Windows

Primero de todo tendremos que configurar el Firewall de la máquina para la práctica

En este caso lo dejaremos Inactivo pero se recomienda no tocar el estado del firewall.

A continuación iremos a Administrador del servidor>Agregar roles y características>Roles de Servidor para añadir el servidor web

En pantalla se muestra la opción que debemos escoger, escogeremos la opción Servidor web (IIS) para instalar todo lo relacionado con los servicios web.

Tras el paso anterior tendremos que seleccionar los servicios de rol para el Servidor web IIS

Aquí tendremos que seleccionar Servidor FTP que seleccionará automáticamente los servicios inferiores, además se selecciona automáticamente Herramientas de administración y Consola de administración.

Una vez terminemos de configurar los servicios para el Servidor IIS…

Comenzará a instalarse todos los servicios que hemos seleccionado durante la configuración, tendremos que esperar unos minutos a que termine.

Una vez tengamos instalado todo correctamente haremos una comprobación en el navegador escribiendo en la barra de búsqueda la dirección http://localhost viendo la página por defecto de nuestra web

El contenido de nuestra página web se publicará en la carpeta C:\inetpub\wwwroot donde subiremos nuestros documentos, carpetas, imágenes…

Comprobamos ahora los servicios activos para poder ver cual es el que nos permite poder publicar y visualizar nuestras páginas web

Haremos WIN+R>services.msc>Servicio de publicación de World Wide Web y como vemos el servicio se ejecuta según se inicia el equipo y comprobamos que está en ejecución.

Ahora vamos a crear una carpeta apuntes para nuestra página web, la cual servirá como una nueva página en la que subiremos los archivos que queramos

En nuestro caso crearemos aquí los documentos dhcp.txt, dns.txt, ftp.txt (vacíos o con contenido, es indiferente) para poder probar que esta nueva sección de la página web muestra los documentos.

Ahora haremos lo mismo pero crearemos una carpeta examenes para la nueva sección de nuestra página web

También tenemos que crear documentos para la posterior comprobación como examen_dhcp.txt, examen_dns.txt, examen_ftp.txt como se muestra en la captura.

Dentro de la carpeta examenes crearemos un archivo .html el cual será el “código” de la página web para mostrar el contenido

El Servidor IIS va a buscar este archivo inicio.html para mostrar su código en pantalla al usuario que busque nuestra página web, pondremos un título sencillo para las comprobaciones.

Haremos lo mismo que hicimos en los pasos anteriores con una carpeta recuperaciones como se muestra en la captura

Esta carpeta será parte de la página web como la nueva sección de recuperaciones.

Por último como vemos en la captura copiamos los archivos examen_(…).txt en esta carpeta recuperaciones…

Como se muestra ahora en la captura en vez de crear inicio.html crearemos el archivo index.html para que muestre el contenido de la página web.
