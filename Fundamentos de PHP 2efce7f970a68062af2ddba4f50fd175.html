Clasificación de los lenguajes de guiones

La clasificación de los tipos de páginas web es:

  • Páginas estáticas: una página es estática cuando su contenido no puede ser modificado ni desde el servidor que la aloja ni desde el cliente ni programa.
<body>
				Hoy es 21/01/2026 y son las 09:09:30 de la mañana
</body>
  • Páginas dinámicas: una página es dinámica cuando su contenido puede ser modificado, bien de forma automática o bien mediante la intervención del usuario. Para hacer esto debemos tener un script con un conjunto de instrucciones en un lenguaje determinado, por ejemplo un código HTML.
<html>
<head>
<script type="text/javascript">
			var son= new Date();
			var fecha=son.getDate()+" - "+(son.getMonth()+1)+"-"+son.getFullYear(); 
			var hora=son.getHours()+":"+son.getMinutes()+":"+son.getSeconds();
			document.write('Hoxe é '+fecha+' e son as '+hora+' horas');
</script>
</head>
</body>
</html>

Los lenguajes de guiones pueden clasificarse en:

  • Lenguajes de guión del lado del cliente: pueden ser interpretados directamente por el navegador como HTML, CSS, VBS…
  • Lenguajes de guión del lado del servidor: son reconocidos, ejecutados e interpretados por el servidor y se envían en un formato comprensible al cliente como PHP, PERL…

PHP

PHP (PHP Hypertext Preprocessor) es un lenguaje de código abierto, interpretada, de alto nivel, incrustada en páginas HTML y ejecutada en un servidor. Creado en 1994 ya han aparecido distintas versiones.

Para ejecutar PHP necesitamos un servidor que soporte PHP. A continuación escribimos los archivos PHP (.php) y los guardamos en el servidor. Podemos instalar y configurar un servidor web como Apache, un intérprete de PHP y una base de datos MySQL por separado o en un paquete que instale todo junto.

1. Herramientas de edición de código

Se puede utilizar cualquier editor de texto para escribir código PHP pero no se recomienda utilizar editores de texto avanzados (Word, Libreoffice…) al añadir caracteres especiales no comprendidos por el navegador. Los editores recomendados son Notepad++, Bloc de notas, VS Code…

2. Integración con los lenguajes de marcas

Existen distintas formas de incrustar PHP dentro de una página HTML:

  • Delimitando las instrucciones PHP dentro de los caracteres:
<?php
<?
  • Utilizando etiquetas cortas:
<?
?>

IMPORTANTE: Activar directiva short_open_tag en php.ini
